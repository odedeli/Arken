# Arken
Arken turns a physical home document cabinet into a secure, paperless archive. It lets a household store scanned and born-digital documents - bills, contracts, certificates, manuals, statements, IDs - in one structured, searchable place, with their properties (metadata) tracked and the whole collection encrypted at rest.
